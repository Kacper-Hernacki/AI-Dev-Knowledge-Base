## LangChain Lesson (Node + TypeScript)

- Install: `npm install`
- Dev: `npm run dev`
- Start: `npm run start`
- Typecheck: `npm run typecheck`
- Test: `npm test`
- Run agent lesson: `npm run lessons:agents`

> Requires Node.js 20 or newer so that `AbortSignal.any` is available for LangChain.

### Env

This project includes a `.env/` directory. Create `.env/.env` and add your variables, e.g.:

```
PORT=3000
```
